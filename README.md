# Restaurant Concept Portfolio

A premium, high-fidelity restaurant showcase featuring two distinct dining experiences built from scratch using modern semantic HTML5, vanilla CSS3, and JavaScript (ES6+).

## 🚀 Live Demos

* 🚪 **[Showcase Portal](https://uzaykacargil.github.io/sushi-restaurant-website/)** — Select and enter either concept from a custom entry page.
* 🍣 **[SushiZen Fine Dining](https://uzaykacargil.github.io/sushi-restaurant-website/sushizen/)** — Elegant, dark-luxury fine dining concept.
* 🍜 **[YATAI Street Food](https://uzaykacargil.github.io/sushi-restaurant-website/yatai/)** — High-energy street food and online ordering concept.

---

## 🍣 Concept A: SushiZen Fine Dining

**Live Link:** [Launch SushiZen](https://uzaykacargil.github.io/sushi-restaurant-website/sushizen/)

SushiZen is designed for premium, upscale dining establishments. It features a dark, meditative aesthetic with elegant gold accents, tailored to inspire trust, highlight chef artistry, and drive physical table bookings.

### ✨ Key Features
* **Interactive Table Reservations:** A validation-guarded table booking modal with customized selections for date, time slot, and guest count.
* **Dynamic Menu Filtering:** Smoothly filters through categories (Nigiri, Signature Rolls, Sashimi, Desserts) without reloading the page.
* **Premium Brand Storytelling:** Sections highlighting the dining philosophy, daily imports, sake pairing guides, and tatami private rooms.
* **Modern Aesthetics:** Features gold accent typography, glassmorphism containers, custom-timed fade-in reveal animations, and high-fidelity AI-generated assets.

## 🍜 Concept B: YATAI Street Food

**Live Link:** [Launch YATAI](https://uzaykacargil.github.io/sushi-restaurant-website/yatai/)

YATAI is a neon-lit, urban street food concept designed around speed and pickup/delivery options. It is optimized for mobile-first, high-traffic casual dining.

### ✨ Key Features
* **Active Cart Drawer:** A sliding sidebar cart that manages order additions, quantities, subtotals, and delivery fees in real-time.
* **Delivery & Pickup Toggle:** Instantly switches the cart behavior, modifying fields and fees based on the delivery method.
* **Checkout Flow:** An interactive checkout modal requiring customer details, delivery address (conditional), and preferred delivery time slots, ending with a checkout success screen.
* **High-Energy Branding:** Dynamic marquee announcement ticker, bold typography combinations (Bebas Neue & Inter), neon orange/cyan accent lighting, and custom hover states.

---

## 🛠️ Tech Stack & Design System
* **Core Languages:** Semantic HTML5, Vanilla CSS3 (utilizing modern CSS variables and flexbox/grid layouts), and Vanilla JavaScript (ES6+).
* **Typography:** Curated Google Fonts integration:
  * *SushiZen:* Playfair Display (editorial elegance) & Outfit (sleek geometric sans-serif).
  * *YATAI:* Bebas Neue (bold, urban headlines) & Inter (ultra-readable interface copy).
* **Performance:** Mobile-responsive design, native lazy loading on menu images, and clean modular structures without heavy frameworks.

## 💻 Running Locally
To run either website locally, simply clone the repository and open the index files:

```bash
# Clone the repository
git clone https://github.com/uzaykacargil/sushi-restaurant-website.git

# Navigate into the project
cd sushi-restaurant-website

# Open the portal index page in your default browser
start index.html
```
